# ROBO

Raspberry Pi robotics project (SunFounder Robot HAT, mecanum wheels, sensors, fan control).  

Documentation is be located at [Wiki](https://github.com/tomoxm/ROBO/wiki)

Build:
- 4x motor's with mecanum wheels
- Sensor suite: 3x laser (TOF) sensors, flame sensor, grayscale sensor, gyroscope sensor
- Fan controlled by MOSFET
- coded in python + n8n blocks
